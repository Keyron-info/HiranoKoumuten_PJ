## 1. はじめに

### 1-1. システム概要

本システムは、協力会社からの請求書提出〜社内承認〜支払い管理までを
クラウド上で一元管理する請求書管理システムです。

### 1-2. 役職別にできること

| 機能 | 現場監督 | 役員 | 経理 | 管理者 |
|------|:-:|:-:|:-:|:-:|
| 請求書の閲覧 | 担当現場のみ | 全件 | 全件 | 全件 |
| 承認・却下・差し戻し | ○（第一承認） | ○ | ○（最終確認） | ○ |
| 赤ペン修正 | × | × | ○ | ○ |
| 月次締め処理 | × | × | ○ | ○ |
| 支払い表・PDF出力 | × | × | ○ | ○ |
| ユーザー管理・マスター管理 | × | × | ○ | ○ |
| 監査ログ閲覧 | × | × | ○ | ○ |

### 1-3. 承認フローの全体像

```
協力会社が提出
  → ① 現場監督 承認
  → ② 部長 承認
  → ③ 専務 承認
  → ④ 社長 承認
  → ⑤ 常務 承認
  → ⑥ 経理 確認
  → 支払い処理 → 支払い完了
```

- 各ステップの承認者には自動で通知が届きます。
- 承認操作は取り消せません。
- 途中で差し戻しが発生すると、協力会社と**それまでに承認した全員**に通知が届きます。

## 2. ログイン・アカウント管理

### 2-1. ログインする

1. システムURLにアクセスします。
2. **メールアドレス**（ログインID）と**パスワード**を入力します。
3. 「ログイン」をクリックすると、ダッシュボードが表示されます。

### 2-2. パスワードを忘れた場合

1. ログイン画面の「パスワードを忘れた方はこちら」をクリックします。
2. 登録済みメールアドレスを入力して送信します。
3. 届いたメールのリンクから新しいパスワードを設定します。

### 2-3. パスワードを変更する

1. 画面右上の自分の名前（アイコン）をクリックします。
2. マイプロフィール画面で「パスワード変更」を行います。
3. 現在のパスワードと新しいパスワード（8文字以上）を入力して保存します。

---

## 3. ダッシュボード

ログイン直後に表示されるホーム画面です。

| 表示項目 | 内容 |
|---------|------|
| 承認待ち | 承認フロー進行中の請求書件数 |
| 自分の承認待ち | 自分の承認操作が必要な件数 |
| 協力会社数 | 有効な協力会社の登録数 |
| 今月の支払額 | 当月支払予定の合計金額 |
| 提出済み合計 | 提出済み・承認待ち請求書の件数と合計金額 |
| 最近の請求書 | 直近の請求書リスト |
| 予算消化状況 | 現場別の予算アラート |

## 4. 請求書一覧・詳細

### 4-1. 請求書を探す

1. メニュー「請求書一覧」をクリックします。
2. 以下の条件で絞り込みできます。
   - キーワード検索（請求書番号・会社名）
   - ステータス（下書き/提出済み/承認待ち/承認済み/支払済み/差し戻し/却下）
   - 工事現場 / 協力会社
   - 日付範囲・金額範囲（電子帳簿保存法対応の詳細検索）
3. 行をクリックすると詳細画面が開きます。

### 4-2. 閲覧期間の制限

経理・役員以外のユーザーは、**作成から30日以内**の請求書のみ閲覧できます。

### 4-3. 請求書詳細で確認できること

- 請求明細（品名・数量・単価・金額・消費税）
- 承認履歴（誰がいつ承認したか）
- 現在の承認ステップ
- コメント（@ユーザー名 でメンション可能）
- 修正履歴（赤ペン修正があった場合）

## 5. 承認業務（役員・現場監督）

### 5-1. 自分の承認待ちを確認する

1. メニュー「承認待ち」をクリックします（ダッシュボードのバッジからも遷移可）。
2. 自分の承認が必要な請求書が一覧表示されます。

### 5-2. 承認する

1. 請求書を開き、内容を確認します。
   - 確認ポイント: 工事名・金額が注文書と一致するか・工種・作業期間
2. 「承認する」をクリックします。
3. 次の承認者へ自動通知されます。

> ⚠️ 承認は取り消しできません。

### 5-3. 却下する

1. 「却下」をクリックします。
2. 却下理由を入力して確定します。
3. 協力会社・経理に通知されます。

### 5-4. 差し戻す

1. 「差し戻し」をクリックします。
2. 差し戻し理由・修正指示を具体的に入力します。
3. 協力会社と、それまでに承認した承認者全員に通知されます。

## 6. 赤ペン修正・差し戻し（経理）

協力会社の請求書に誤りがある場合、経理側で修正できます。

### 6-1. 修正手順

1. 請求書詳細を開き、「修正」ボタンをクリックします。
2. 修正する項目（金額・数量・単価・品名）を入力します。
3. **修正理由を必ず入力**して保存します。
4. 協力会社へ差し戻し通知が自動送信されます。
5. 協力会社が「修正内容を承認する」を押すと、承認フローが再開します。

### 6-2. 修正できる期間

請求書の**受領から2日以内**です。期限を過ぎると修正ボタンは表示されません。

> ✅ 修正履歴はすべて監査ログに記録されます。

## 7. 月次締め処理（経理）

### 7-1. 締め日ルール

| 項目 | 内容 |
|------|------|
| 対象期間 | 前月26日〜当月25日 |
| 受付期間（基本） | 当月26日〜末日 |
| 締め処理の実施 | 毎月25日以降 |

### 7-2. 締め処理の手順

1. メニュー「管理」→「月次管理」を開きます。
2. 対象月の期間を確認します。
3. 「締め処理を実行」をクリックします。
4. 提出済み請求書が一括で承認フローに移行します。

> ⚠️ 締め処理は元に戻せません（管理者のみ再開可能）。

### 7-3. 特例パスワードの発行

期限後の提出を許可したい場合に発行します。

1. 月次管理または現場管理で対象を開きます。
2. 「特例パスワード」と有効期限を設定します。
3. 協力会社にパスワードを伝えます。協力会社は提出時にこのパスワードを入力することで期限外でも提出できます。

## 8. 支払い表（経理）

1. メニュー「管理」→「支払い表」を開きます。
2. 年月を選択して「表示」をクリックします。
3. 承認済み請求書から、**現場別・業者別の支払い表**が自動生成されます。

| 列 | 内容 |
|----|------|
| 請求金額 | 業者ごとの請求合計 |
| 安全衛生協力会費 | 請求額10万円以上の場合、請求額の3/1000を自動計算 |
| 差引支払額 | 請求金額 − 協力会費 |

「印刷」ボタンで印刷できます。

## 9. PDF出力・CSV出力（経理）

### 9-1. PDF出力

請求書詳細画面の「PDF出力」をクリックします。
**経理担当者・スーパー管理者のみ**利用できます。

### 9-2. CSV出力

請求書一覧などのCSVエクスポートボタンから出力できます。
会計ソフト（建設大臣等）への取り込みに利用してください。

## 10. ユーザー管理（経理・管理者）

### 10-1. ユーザーを追加する

1. メニュー「管理」→「ユーザー管理」を開きます。
2. 「＋新規ユーザー」をクリックします。
3. 以下を入力します。
   - ユーザー種別（社内ユーザー / 協力会社ユーザー）
   - 姓・名
   - ユーザー名・メールアドレス
   - パスワード（8文字以上）
   - 役職（社内ユーザーの場合。承認フローに影響します）
4. 「ユーザーを作成」をクリックします。

### 10-2. ユーザーを編集・無効化する

- 鉛筆アイコン: 氏名・メール・役職・パスワードの変更
- 電源アイコン: 有効/無効の即時切り替え（退職者などはこちら）

## 11. 協力会社の登録申請の承認（経理）

協力会社が新規登録すると、承認待ちの申請が届きます。

1. メニュー「管理」→「登録申請」を開きます。
2. 申請内容（会社情報・担当者・**振込先金融機関情報**）を確認します。
3. 「承認」をクリックすると、ユーザーが自動作成され、
   初期パスワードが申請者へメールで送信されます。
4. 問題がある場合は「却下」を選び、理由を入力します。

> ✅ 承認時に振込先銀行情報は協力会社マスターに自動登録されます。

## 12. 協力会社管理（経理）

メニュー「管理」→「協力会社管理」から、以下を登録・編集できます。

- 会社名・ふりがな・業種
- 代表者名・連絡先・住所
- インボイス番号（T+13桁）
- 振込先銀行口座（銀行名・支店名・口座番号）

## 13. 工事現場管理（経理）

メニュー「管理」→「現場管理」から登録・編集します。

| 項目 | 説明 |
|------|------|
| 現場名・工事コード | 工事コードは未入力で自動採番 |
| 担当現場監督 | 承認フローの第一承認者になります（**必須**） |
| 現場パスワード | 協力会社が請求書作成時に入力するパスワード |
| 予算金額 | 消化率80%/90%/100%で現場監督に自動アラート |
| 完成・打ち切り | 設定すると新規請求書の作成不可になります |

> ⚠️ 現場監督が未設定の現場には請求書を提出できません。必ず設定してください。

## 14. 工種管理（経理）

メニュー「管理」→「工種管理」で、請求書に使う工種マスター（直接仮設工事・土工事など35種類）を管理します。使用頻度の高い工種は自動的に上位表示されます。

## 15. カレンダー管理（経理）

メニュー「管理」→「カレンダー管理」で、月ごとの支払日・締め日を登録します。
登録した支払日は支払い表・ダッシュボードに反映されます。

---

## 16. 監査ログ（経理）

メニュー「管理」→「操作ログ」で、システム上の全操作履歴を確認できます。

- 記録対象: 作成・更新・削除・承認・却下・差し戻し・ログイン・PDF出力・締め処理 等
- ユーザー名・キーワード・対象（請求書/ユーザー/現場等）で検索可能
- 電子帳簿保存法対応のため、ログの改ざん・削除はできません

## 18. AIヘルプチャットボット

## 19. よくある質問（FAQ)

**Q. 承認を間違えて押してしまった**
A. 承認は取り消せません。次の承認者に連絡して却下または差し戻しを依頼してください。

**Q. 協力会社から「提出できない」と連絡があった**
A. ①受付期間外（基本: 26日〜末日）②締め処理済み ③現場監督未設定 ④協力会社情報未紐付け のいずれかです。期間の問題なら特例パスワードを発行してください。

**Q. 請求書一覧に何も表示されない**
A. 自分のアカウントに所属会社が設定されていない可能性があります。管理者に連絡してください。

**Q. 支払い表に請求書が出てこない**
A. 支払い表には「承認済み・支払い準備中・支払い済み」の請求書のみ表示されます。承認フロー進行中の請求書は含まれません。

**Q. 退職者のアカウントはどうすればよい？**
A. ユーザー管理から「無効化」してください。削除はできません（履歴保持のため）。

## 20. トラブルシューティング

| 症状 | 対処 |
|------|------|
| ログインできない | パスワードリセットを実施。解決しなければ管理者がユーザー管理で確認 |
| 画面が真っ白・エラーが出る | ブラウザを再読み込み（Cmd/Ctrl+R）。改善しなければKEYRONへ連絡 |
| メール通知が届かない | 迷惑メールフォルダを確認。メールアドレスの登録ミスがないかユーザー管理で確認 |
| PDF出力ボタンがない | 経理・管理者権限が必要です |
| 承認ボタンが表示されない | 自分が現在の承認ステップの担当者か確認してください |
